# 论文证明

请根据输入的信息查找 出发表载体、卷期出版年份、论文标题、人才姓名、致谢。
1、发表载体、卷期出版年份、论文标题、人才姓名等信息通常在首页需要你额外关注一下。
2、致谢内容通常出现在论文正文的最后，通常有对应的关键词。
3、新建条目，内容为 "该证明为项目相关成果发表论文，可作为侧面证明"，box为 [0,0,200,100]，页码应该和致谢的页码一致。

# 注意事项
1、人才姓名
    - 论文通常有多个作者，请勿吧多余的人识别了
    - 坐标需要精准的图片/文件中人名进行框选。

2、 卷期出版年份
    - 卷期出版年份并不是论文的收录时间请注意查看。
    - 通常会在页眉 页脚的位置。
3、 发表载体
    - 内容前面需要加上 "发表载体"四个字
    - 示例:  信息是xxx ，输出   "发表载体: xxxx"
4、 论文标题 必须翻译为简体中文。

5、当前页面如果是需要查找 【致谢】内容，严谨识别其他内容。
